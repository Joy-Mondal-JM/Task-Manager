# Task Management App

## Setup Instructions
1. **Database:** Create a PostgreSQL database `taskdb` and run the schema script provided.
2. **Backend:**
   - `cd task-backend`
   - `npm install`
   - Create `.env` file with your DB credentials.
   - `node server.js`
3. **Frontend:**
   - `cd task-frontend`
   - `npm install`
   - `npm start`
